# Bigtree

High performance hierarchical data rendering based on nested set model (can also be used for adjacency).

## Example Usages
1. Include library
    
    ```xml
    <script src="jquery.js"></script>
    <script src="jquery.bigtree.js"></script>
    ```
2. Usage
    
    ```javascript
    <div id="tree"></div>
    <script>
        $('#tree').bigtree();
    </script>
    ```
